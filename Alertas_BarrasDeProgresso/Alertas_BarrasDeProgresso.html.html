O "& times;" cria o simbolo X. Poderia ser escrito também. A classe "close" é responsável por redirencionar o X para o lado direito. Para criar a funcionalidade que ao clicar no X ele fecha é responsabilidade do "data-dismiss="alert"" + class="alert-dismissible" na div que contém todo o conteúdo que se deseja "fechar".
