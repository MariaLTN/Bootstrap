Sistemas de Grid

As colunas são proporcionais ao tamanho da tela; Ao usar uma "row" temos uma flexbox; Os "col-6", "col-sm-2", "col-md-4-", "col-sm-12" São para formula novos breakpoints para o layout baseado no tamanho do dispositivo. Observação: O sistema de grid respeita o tamanho máximo do grid que é 12, caso de mais do que isso ele quebra e vai para baixo.
